# HYDRYS — Thème Shopify Online Store 2.0

Ce dépôt contient un thème Shopify OS 2.0 premium en français pour HYDRYS, centré sur la filtration d’eau moderne, la conversion, le SEO et l’AI discoverability.

## Structure livrée

- `layout/` : layout principal du thème.
- `templates/` : templates JSON pour l’accueil, produit, collection, catalogue, panier, blog, article, pages de contenu et 404.
- `sections/` : sections dynamiques et blocs configurables pour Theme Editor.
- `snippets/` : snippets de données structurées (`Product`, `FAQPage`).
- `assets/` : CSS global + JS d’animations légères.
- `config/` : schéma de réglages thème + données par défaut.
- `locales/` : traduction française par défaut.
