## Structure

- FAQ accordion with essential information.

## Getting started
Open `index.html` in a browser. Ensure internet access for fonts, images, analytics scripts, and Stripe payment links.

## Configuration
- Replace placeholder Stripe Payment Link URLs in `script.js` with production links.
- Update Google Analytics 4 measurement ID in `index.html` (`G-XXXXXXXXXX`).
- Update TikTok pixel ID (`TT-XXXXXXX`).
- Swap placeholder visuals with final renders in `index.html`.
- Adjust policy/support links in the footer as needed.

## Deployment
Upload the static files (`index.html`, `styles.css`, `script.js`) to your hosting platform (Framer, Webflow custom code embed, Vercel static site, etc.).
